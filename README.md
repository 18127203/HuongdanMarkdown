HuongdanMarkdown
================

Hướng dẫn sử dụng Markdown
Các Thẻ Tiêu Đề: h1, h2, h3, h4…

Markdown sử dụng kí tự # để bắt đầu cho các thẻ tiêu đề, dựa vào số lượng kí tự # liền nhau sẽ hiển thị ra HTML tương ứng.

# Thẻ H1

## Thẻ H2

### Thẻ H3

#### Thẻ H4

Đây là thẻ H1

Đây là thẻ H2

Đây là thử H3
